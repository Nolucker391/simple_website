Для реализации функциональности увеличения окна при нажатии на фотографию в HTML, вы можете использовать JavaScript и CSS. Вот пример решения:  1. Создайте HTML-разметку, включающую фотографию и окно для увеличения:  ```html <div class="image-container"> <img src="path/to/your/image.jpg" alt="Фотография" onclick="zoomIn()"> </div>  <div id="zoomed-in-image" onclick="zoomOut()"> <img src="path/to/your/image.jpg" alt="Увеличенная фотография"> </div> ```  2. Добавьте CSS-стили для контейнера фотографии и окна увеличения:  ```css .image-container { width: 200px; height: 200px; }  #zoomed-in-image { display: none; position: fixed; top: 0; left: 0; width: 100%; height: 100%; background-color: rgba(0, 0, 0, 0.8); text-align: center; z-index: 9999; }  #zoomed-in-image img { max-width: 80%; max-height: 80%; margin-top: 10%; } ```  3. Добавьте JavaScript-функции для управления отображением окна увеличения:  ```javascript function zoomIn() { document.getElementById("zoomed-in-image").style.display = "block"; }  function zoomOut() { document.getElementById("zoomed-in-image").style.display = "none"; } ```  Теперь, при нажатии на фотографию, окно увеличения будет отображаться, а при нажатии на увеличенную фотографию окно будет скрываться. Вы можете настроить стили и размеры окна увеличения в соответствии с вашими потребностями.
